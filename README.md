# Entering Esan

Entering Esan is a mobile-friendly static website for a university digital humanities project about Esan culture. It uses only plain HTML, CSS, and JavaScript, and it is ready for free GitHub Pages hosting.

## Files

- `index.html` contains the page structure, navigation, content sections, and reflective entry modal.
- `styles.css` contains the responsive layout, palette, typography, cards, modal, and accessibility styles.
- `script.js` contains the client-side menu and modal behavior.
- `assets/entering-esan-hero.png` is the local hero image used by the homepage.
- `assets/wedding/` contains optimized local copies of the Esan traditional wedding photographs used in the marriage section.
- `assets/culture/` contains the Esan map image and masquerade photographs used in the culture sections.

## Privacy Note

The entry prompts run only in the visitor's browser. The site has no backend, does not use analytics, and does not collect, save, or send visitor responses. The form data is read only to complete the entry interaction and is then discarded.

The YouTube resource is not loaded automatically. Visitors must choose **Load Video** before the page creates the embedded YouTube frame.

## Editing Content

The current cultural content includes supplied course notes and some areas marked for future expansion. Before using the site beyond a course prototype, review and expand the material with appropriate scholars, community contributors, and governance processes.
